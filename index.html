La Toolbox : La Registry comme écrin

Qu'est-ce que la Registry ?


  1. Projet Open source Distribution sous licence Apache V2
  2. Un lieu central ou stocker et distribuer des images Docker
  3. Stocke les différentes couches d'une image
  4. Stocke la description permettant de construire une image
  5. Expose une API aux clients Docker
  6. En V2 emet des notifications webhook (autoredeploy )

  7. Docker Hub: registry SaaS (images publiques gratuites )
  8. Docker Trusted Registry: version on-premise de Docker Hub
